# Основы FastApi для Backend

## Домашнее задание Pro

1. Добавить в модель Note поле `created_at` - дата создания заметки
2. Добавить в промпт текущую дату.
3. Добавить в промпт возможность генерации запроса, который вернет заметки в заданном интервале времени, например - покажи заметки за прошлый месяц, покажи заметки за январь 2026 года.

Подсказка по добавлению поля. После создания миграции её нужно отредактировать:
```
def upgrade() -> None:
    op.add_column(
        'note',
        sa.Column('created_at', sa.DateTime(timezone=True), nullable=True)
    )
    # Обновляем старые записи
    op.execute("UPDATE note SET created_at = CURRENT_TIMESTAMP")

def downgrade() -> None:
    op.drop_column('note', 'created_at')
```

4. Вышлите скриншот выполнения тестового запроса "Верни все мои заметки за текущий месяц".
